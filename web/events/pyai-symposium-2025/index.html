PyAI Symposium 2025

2025년 11월 30일
13:00 - 18:00
마이크로소프트 광화문 13층
이벤트 페이지 바로가기

PyAI Symposium 2025는 파이썬을 중심으로 한 인공지능 기술과 실제 적용 사례를 공유하는 자리입니다.
학생, 연구자, 개발자, 실무 전문가가 함께 모여 최신 기술 동향을 살펴보고 경험을 나누는 시간을 갖습니다.

📆 행사 개요

  • 일시: 2025년 11월 30일 (일) 오후 1시–6시
  • 장소: 서울 종로구 종로1길 50 더케이트윈타워 A동 13층 한국마이크로소프트
    • 카카오맵 / 네이버지도
    • 본 행사 건물의 주차비 지원은 불가합니다. 대중 교통을 이용하시길 권장드립니다

🐍 행사 내용

  • 파이썬 기반 LLM 기술 소개
    • Python 생태계를 활용한 모델 구축, 실험, 운영 환경 구성 사례를 소개합니다.
  • LLM 활용 및 응용 사례
    • 프롬프트 설계, 파인튜닝, 에이전트 시스템 구현 등
    • 다양한 LLM 응용 경험과 실제 서비스 적용 사례를 공유합니다.
  • 오픈소스 도구 및 프레임워크
    • LangChain, FastAPI, PyTorch, Hugging Face 등
    • 파이썬 기반 AI 개발 도구와 워크플로우를 실무 관점에서 다룹니다.
  • 운영 및 MLOps 경험
    • 모델 배포, 성능 최적화, 모니터링 등
    • LLM 운영 과정에서의 실무 노하우를 논의합니다.
  • 네트워킹
    • 파이썬과 LLM 분야 종사자 간 기술 질문과 정보 교류를 위한 자리입니다.
  • 오픈스페이스

🗣️ 세션 시간표

  1. $5짜리 프롬프트로 $2613짜리 취약점 찾은 썰

🎙️ 연사: 윤석찬 (Airflow Security Team, Apache Software Foundation)

DEF CON 33에서 목격한 LLM의 압도적인 잠재력을 계기로, 프롬프트와 ChatGPT·Codex만으로 Django와 FastAPI에서 실제 CVE를 찾아낸 여정을 상세히 공유합니다.

단순한 해킹 스토리가 아니라,

  • 프롬프트 엔지니어링으로 어떻게 보안 탐색을 유도할 수 있는지
  • LLM이 기존 보안 워크플로우를 어떻게 재편하고 있는지
  • Offensive security가 맞이한 새로운 패러다임이 무엇인지

를 실전 사례 중심으로 살펴봅니다.


2. 고등학생인 내가 MLOps라니 무리무리!! (무리가 아니었다?!)
🎙️ 연사: 윤지상 (서울디지텍고등학교)

“학생 개발자가 기업에서 LLMOps 플랫폼을 구축한다면?” 중학생 시절 프리랜서로 시작해, 현재는 실제 기업에서 학습근로자로 AI 플랫폼을 개발 중인 연사가 자신의 경험을 가감 없이 공유합니다.

FastAPI와 Pydantic으로 타입 안정성을 확보하고, MLflow로 Kubernetes 환경에서 학습 재현성을 구현하며, LLMOps 파이프라인 전체를 손수 설계한 실제 사례를 다룹니다.


3. 모두를 위한 친절한 언어모델(LLM) 학습 가이드

🎙️ 연사: 이준범 (AI GDE / 래블업 연구원)

기업에서 갑자기 “LLM 좀 이해해보세요”라는 미션을 받았지만 어디서부터 시작해야 할지 막막한 분들을 위한 세션입니다.

임베딩, 레이어, GPU 병렬처리, 토크나이저, 행렬 연산 등 LLM을 이루는 핵심 요소들을 수학 공포증이 있는 개발자도 이해할 수 있게 친절하게 풀어냅니다.

이 세션은 단순한 개념 나열이 아니라,

  • 왜 이 구조가 필요한지
  • 어떤 문제가 생기고 어떻게 해결되는지
  • 실제 업무에서 어떤 부분이 도움이 되는지

까지 이어지는 실무 중심 가이드입니다. “그냥 코딩을 잘하고 싶은데 LLM이 자꾸 등장해서 부담되는” 분들에게 확실한 방향성을 제시하는 시간입니다.

  1. 당신도 할 수 있다 — 개인 생산성을 위한 AI Agent 개발기 (with Pydantic AI)

🎙️ 연사: 배권한 (LINE)

Pydantic AI와 OpenRouter를 활용해 개인의 생산성을 극적으로 끌어올린 AI 비서를 만드는 과정을 실제 사례와 함께 공개합니다.

채팅 흐름을 설계하고, 외부 Tool을 연동하며, MCP 기반 확장을 적용하는 방법까지 ‘개인용 AI Agent를 어떻게 구축할 수 있는가’를 매우 현실적인 관점에서 설명합니다.

에이전트의 역할 정의부터 모델 선택 기준, 운영 시 주의사항까지, 누구나 실생활에 바로 적용할 수 있는 실전 팁을 얻을 수 있습니다.

  1. Retrieval 모델은 무엇이고 어떻게 학습할까?

🎙️ 연사: 유대곤 (Sionic AI / Sentence-Transformers Contributor)

RAG 시스템의 성능을 결정짓는 핵심인 Retrieval 모델이 어떻게 작동하고 학습되는지, 그리고 왜 이 기술이 LLM 기반 서비스에 필수인지 실전 중심으로 살펴봅니다.

한국어 Retrieval SoTA 모델을 개발하며 얻은 실제 경험을 바탕으로,

  • 데이터 구성 방법
  • 학습 프레임워크 설계
  • 모델 개선 전략
  • 커뮤니티 생태계(Hugging Face 등)에서의 최신 트렌드

까지 폭넓게 다룹니다.
